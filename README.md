# Robo Eyes Emotions

An Arduino Nano project that gives life and emotions to an OLED display using the [FluxGarage RoboEyes](https://github.com/FluxGarage/RoboEyes) library and Adafruit SSD1306.

The robot face reacts randomly with various moods such as *angry*, *tired*, *confused*, *happy*, *dead*, and more.


## Features
- Random emotional reactions and animations
- Idle blinking and flickering effects
- Custom messages and facial expressions
- Uses `Adafruit_GFX` + `Adafruit_SSD1306` for display control
- Compatible with Arduino Nano and similar boards


## Hardware Requirements
- **Arduino Nano** (or Uno)
- **0.96" OLED Display (128x64, I2C, SSD1306)**
- Jumper wires  
- (Optional) Breadboard


## Libraries Used
- [FluxGarage RoboEyes](https://github.com/FluxGarage/RoboEyes)  
- [Adafruit SSD1306](https://github.com/adafruit/Adafruit_SSD1306)  
- [Adafruit GFX](https://github.com/adafruit/Adafruit-GFX-Library)
